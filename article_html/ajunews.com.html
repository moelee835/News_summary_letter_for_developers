오픈AI, 'GPT-5.3 인스턴트' 출시… 정확도 향상·환각 감소·대화개선

김성현 기자입력 2026-03-04 08:43
  • 기사공유
  • 폰트크기


오픈AI가 챗GPT에서 가장 널리 사용되는 모델인 GPT-5.3 인스턴트의 업데이트를 3일(현지시간) 공개했다.
 
이번 업데이트로 GPT-5.3 인스턴트는 더 정확한 답변을 제공하며, 웹 검색 시 맥락을 반영한 결과를 제시해 대화 흐름을 개선했다는 것이 회사 측의 설명이다. 특히 질문의 의도를 파악해 관련성 높은 정보를 우선적으로 제시하고, 응답의 자연스러움을 유지하도록 설계됐다.
 
웹 정보 활용 방식도 향상됐다. 온라인 자료와 모델의 기존 지식을 효과적으로 결합해 단순 요약을 넘어 최근 이슈의 맥락까지 설명한다.

정확성 측면에서는 환각 현상이 줄었다. 오픈AI 내부 평가 결과, 법률·금융 등 고위험 분야에서 웹 검색 사용 시 환각률이 26.8%, 내부 지식만 사용할 경우 19.7% 감소했다. 비식별 사용자 피드백 기반 평가에서는 웹 사용 시 22.5%, 웹 미사용 시 9.6% 줄어든 것으로 확인됐다.
 
대화 스타일도 조정됐다. 이전 모델에서 나타나던 불필요한 답변 거절, 과도한 감정 표현, 사용자 의도 추정 등을 줄여 핵심에 집중한 응답을 제공한다.
 
글쓰기 지원 기능은 소설 초안 작성, 문장 수정, 아이디어 탐색 등 창작 작업에서 더 유용하게 활용될 수 있도록 강화됐다.
 
GPT-5.3 인스턴트는 현재 챗GPT 모든 사용자에게 적용되며, 개발자용 API에서는 gpt-5.3-chat-latest 모델로 제공된다. 씽킹 모델과 프로 모델에 대한 업데이트는 순차적으로 진행될 예정이다.
 
한편 기존 GPT-5.2 인스턴트는 유료 사용자 대상으로 모델 선택기 내 레거시 모델 섹션에서 3개월간 유지되며, 오는 6월 3일 서비스가 종료된다.
