    엠클라우드브리지, 대화형 분석 ‘Ai 365 데이터 에이전트’로 차세대 AI+BI 시장 공략

    AI 통합 업무 플랫폼 기업 엠클라우드브리지(대표 이혁재)는 자연어 기반 데이터 분석과 생성형 BI, 대화형 데이터 분석 수요 확대에 맞춰 자사의 대화형 분석 에이전트 ‘Ai 365 데이터 에이전트(Ai 365 Data Agent)’를 중심으로 AI+BI 시장 공략을 강화한다고 밝혔다.

    엠클라우드브리지에 따르면 Ai 365 데이터 에이전트는 기업 데이터 환경 위에 자연어 분석, 설명형 인사이트, 업무 자동화를 결합해 대화형 데이터 분석 체계로의 전환을 지원하는 솔루션이다. 회사는 해당 솔루션과 교육·컨설팅 역량을 결합해 국내 기업의 기존 BI 환경을 차세대 AI+BI 체계로 단계적으로 고도화한다는 전략이다.

    최근 비즈니스 인텔리전스(BI)·애널리틱스 시장은 정적 대시보드와 수동 보고 중심 구조에서 벗어나 AI 데이터 분석과 생성형 BI, 대화형 분석이 결합된 형태로 빠르게 변화하고 있다.

    글로벌 리서치 기관인 가트너(Gartner)와 포레스터(Forrester) 등은 증강 분석(Augmented Analytics), 생성형 AI, 에이전트 기반 AI(Agentic AI)가 결합되면서 사용자가 자연어로 질문하고 시스템이 분석과 후속 조치를 수행하는 분석 환경이 확산될 것으로 전망하고 있다.

    Ai 365 데이터 에이전트는 Microsoft Fabric, Databricks, Snowflake, Azure, 온프레미스 데이터베이스, 파일 저장소, 업무 SaaS 등 다양한 데이터 소스를 연결해 자연어 기반 질의와 분석, 자동 요약, 보고서 생성, 알림, 업무 자동화까지 수행하도록 설계된 플랫폼이다. 사용자는 “지난 분기 제품군별 매출과 이익률을 요약하고 주요 변동 요인을 정리해달라”는 식의 자연어 질문만으로 데이터 조회와 분석, 시각화, 인사이트 도출 결과를 한 번에 확인할 수 있다.

    주요 기능으로는 자연어 기반 대화형 분석, Agentic AI 기반 분석·실행 프로세스 연결, 도메인별 실무 특화 인사이트 제공 등이 있다. 사용자는 별도의 대시보드를 찾지 않고 자연어 질의를 반복하며 필요한 수준까지 드릴다운 분석을 수행할 수 있으며, Text-to-SQL과 RAG(Retrieval-Augmented Generation) 기술을 통해 다양한 데이터 소스에 대한 질의를 자동 생성하고 결과를 표·차트·요약 보고서 형태로 제공받는다.

    또한 데이터 수집·정제·분석·요약·보고서 작성·알림·업무 자동화 등 분석 이후의 전체 흐름을 AI 에이전트가 역할별로 처리하도록 설계됐으며, Power Automate 등과 연계해 분석 결과를 기반으로 후속 업무까지 자동 실행할 수 있는 점도 특징이다.

    엠클라우드브리지가 진행한 PoC 및 고객 프로젝트에 따르면 Ai 365 데이터 에이전트를 도입한 조직은 정기 보고서와 대시보드 제작에 투입되던 시간을 최소 30% 이상 절감했으며, 현업 부서의 자체 데이터 분석 비중도 증가한 것으로 나타났다.

    이혁재 엠클라우드브리지 대표는 “최근 BI 환경은 단순한 지표 시각화를 넘어 사용자와 데이터가 질문과 응답을 주고받는 대화형 분석 체계로 발전하고 있다”며 “Ai 365 데이터 에이전트를 중심으로 고객사의 데이터·BI 인프라를 차세대 AI+BI 환경으로 고도화하는 데 주력하고 있다”고 말했다.

    한편, 엠클라우드브리지는 데이터·AI·클라우드 전문 기업으로 다양한 글로벌 기술 기업과의 파트너십을 기반으로 데이터 플랫폼 구축, AI 데이터 분석, 클라우드 인프라 및 교육 서비스를 제공하고 있다.
    엠클라우드브리지 대화형 데이터 분석 에이전트 와우넷 오늘장전략 와우퀵 앱 실전주식 투자정보. onAir, Editor's Pick

      2026-03-10 14:27 엠클라우드브리지, 대화형 분석 ‘Ai 365 데이터 에이전트’로 차세대 AI+BI 시장 공략


      AI 통합 업무 플랫폼 기업 엠클라우드브리지(대표 이혁재)는 자연어 기반 데이터 분석과 생성형 BI, 대화형 데이터 분석 수요 확대에 맞춰 자사의 대화형 분석 에이전트 ‘Ai 365 데이터 에이전트(Ai 365 Data Agent)’를 중심으로 AI+BI 시장 공략을 강화한다고 밝혔다.

      엠클라우드브리지에 따르면 Ai 365 데이터 에이전트는 기업 데이터 환경 위에 자연어 분석, 설명형 인사이트, 업무 자동화를 결합해 대화형 데이터 분석 체계로의 전환을 지원하는 솔루션이다. 회사는 해당 솔루션과 교육·컨설팅 역량을 결합해 국내 기업의 기존 BI 환경을 차세대 AI+BI 체계로 단계적으로 고도화한다는 전략이다.

      최근 비즈니스 인텔리전스(BI)·애널리틱스 시장은 정적 대시보드와 수동 보고 중심 구조에서 벗어나 AI 데이터 분석과 생성형 BI, 대화형 분석이 결합된 형태로 빠르게 변화하고 있다.

      글로벌 리서치 기관인 가트너(Gartner)와 포레스터(Forrester) 등은 증강 분석(Augmented Analytics), 생성형 AI, 에이전트 기반 AI(Agentic AI)가 결합되면서 사용자가 자연어로 질문하고 시스템이 분석과 후속 조치를 수행하는 분석 환경이 확산될 것으로 전망하고 있다.

      Ai 365 데이터 에이전트는 Microsoft Fabric, Databricks, Snowflake, Azure, 온프레미스 데이터베이스, 파일 저장소, 업무 SaaS 등 다양한 데이터 소스를 연결해 자연어 기반 질의와 분석, 자동 요약, 보고서 생성, 알림, 업무 자동화까지 수행하도록 설계된 플랫폼이다. 사용자는 “지난 분기 제품군별 매출과 이익률을 요약하고 주요 변동 요인을 정리해달라”는 식의 자연어 질문만으로 데이터 조회와 분석, 시각화, 인사이트 도출 결과를 한 번에 확인할 수 있다.

      주요 기능으로는 자연어 기반 대화형 분석, Agentic AI 기반 분석·실행 프로세스 연결, 도메인별 실무 특화 인사이트 제공 등이 있다. 사용자는 별도의 대시보드를 찾지 않고 자연어 질의를 반복하며 필요한 수준까지 드릴다운 분석을 수행할 수 있으며, Text-to-SQL과 RAG(Retrieval-Augmented Generation) 기술을 통해 다양한 데이터 소스에 대한 질의를 자동 생성하고 결과를 표·차트·요약 보고서 형태로 제공받는다.

      또한 데이터 수집·정제·분석·요약·보고서 작성·알림·업무 자동화 등 분석 이후의 전체 흐름을 AI 에이전트가 역할별로 처리하도록 설계됐으며, Power Automate 등과 연계해 분석 결과를 기반으로 후속 업무까지 자동 실행할 수 있는 점도 특징이다.

      엠클라우드브리지가 진행한 PoC 및 고객 프로젝트에 따르면 Ai 365 데이터 에이전트를 도입한 조직은 정기 보고서와 대시보드 제작에 투입되던 시간을 최소 30% 이상 절감했으며, 현업 부서의 자체 데이터 분석 비중도 증가한 것으로 나타났다.

      이혁재 엠클라우드브리지 대표는 “최근 BI 환경은 단순한 지표 시각화를 넘어 사용자와 데이터가 질문과 응답을 주고받는 대화형 분석 체계로 발전하고 있다”며 “Ai 365 데이터 에이전트를 중심으로 고객사의 데이터·BI 인프라를 차세대 AI+BI 환경으로 고도화하는 데 주력하고 있다”고 말했다.

      한편, 엠클라우드브리지는 데이터·AI·클라우드 전문 기업으로 다양한 글로벌 기술 기업과의 파트너십을 기반으로 데이터 플랫폼 구축, AI 데이터 분석, 클라우드 인프라 및 교육 서비스를 제공하고 있다.
