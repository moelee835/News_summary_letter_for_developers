엠클라우드브리지, 'Ai 365 데이터 에이전트'로 AI+BI 시장 확대
  • 진보현 기자 stoo22@cstimes.com
  • 기사출고 2026년 03월 10일 14시 39분
  • 댓글 0
이 기사를 공유합니다

컨슈머타임스=진보현 기자 | 엠클라우드브리지는 자사의 대화형 분석 에이전트 'Ai 365 데이터 에이전트(Ai 365 Data Agent)'를 앞세워 차세대 AI+BI(비즈니스 인텔리전스) 시장 확대에 나선다.

Ai 365 데이터 에이전트는 기업의 데이터 환경에 자연어 분석, 설명형 인사이트, 업무 자동화 기능을 결합한 플랫폼이다. 기존의 정적 대시보드 중심 분석 체계를 대화형으로 전환하도록 돕는 역할을 한다.

이 솔루션은 마이크로소프트 패브릭을 비롯해 데이터브릭스, 스노우플레이크, 애저 등 다양한 데이터 소스와 연동된다. 사용자가 자연어로 특정 분기의 매출 요약이나 변동 요인 분석을 요청하면 데이터 조회부터 시각화, 인사이트 도출까지의 전 과정을 에이전트가 수행한다.

핵심 기능인 자연어 기반 대화형 분석은 사용자가 별도의 대시보드 조작 없이 질의응답을 반복하며 상세 분석을 진행할 수 있도록 설계됐다. 

또한, Text-to-SQL 및 RAG 기술을 활용해 데이터 수집, 보고서 작성, 알림 발송 등의 업무 흐름을 개별 AI 에이전트가 분담해 처리한다.
